# Joycity Wildberries Dashboard

Streamlit‑дашборд для оперативного мониторинга продаж на Wildberries. Приложение подключается к WB API, собирает данные о продажах, заказах, остатках и карточках товара, расчитывает ключевые показатели и визуализирует их. Доступно сравнение периодов, авто‑инсайты (аномалии), графики динамики, разбор маржи и таблица по артикулам.

## Запуск

1. Клонируйте репозиторий и создайте виртуальное окружение:

   ```bash
   python -m venv venv
   source venv/bin/activate        # Windows: venv\Scripts\activate
   pip install -r requirements.txt
   ```

2. Получите API‑токен WB. В личном кабинете продавца Wildberries перейдите в «Профиль» → «Настройки» → «Доступ к API» и создайте новый токен с категориями «Статистика», «Аналитика», «Контент» и «Продвижение». Скопируйте токен.

3. Создайте файл `.env` на основе `.env.example` и вставьте в него значение `WB_API_TOKEN`.

   ```bash
   cp .env.example .env
   # отредактируйте .env и вставьте ваш токен
   ```

4. Запустите приложение:

   ```bash
   streamlit run app.py
   ```

Приложение откроется на `http://localhost:8501`. В боковой панели можно выбрать период (день, 7 дней, 30 дней), настроить фильтры и обновить данные. По умолчанию данные кэшируются на 15 минут.

## Структура

- **app.py** – главное приложение Streamlit.
- **requirements.txt** – зависимости Python.
- **.env.example** – шаблон переменных окружения.
- **README.md** – этот файл с инструкциями.

## Доработки

- Подключить расходы на рекламу (`ads_per_sku`) через эндпойнт `adv/v2/fullstats` WB API.
- Загрузить себестоимость (`cogs_per_sku`) из вашей учётной системы (CSV, Google Sheets или база данных), чтобы корректно расчитывать маржу.
- Настроить Telegram‑бот для алёртов (например, когда остатки на складе < 10 дней или ДРР > 25 %).
- Подключить PostgreSQL для сохранения истории и сравнения периодов без ограничения по API.
- Разнести код по модулям (`wb/client.py`, `wb/etl.py`, `wb/metrics.py`, `ui/`) для удобства поддержки.
